Terminal Soul Drip

A hyper-fast, minimalist dark theme that feels like opening a wormhole into a hacker bunker in 1997 but with Gen Alpha energy and modern syntax highlighting.

Features

  • Dark charcoal background with warm accents
  • Gruvbox-inspired color palette
  • Monospace fonts only (JetBrains Mono preferred)
  • Strong contrast for readability
  • Zero bloat - no gradients, shadows, or animations

Code Example

// Fresh route handler
export const handler: Handlers = {
  async GET(req, ctx) {
    const terminal = new TerminalSoulDrip({
      theme: 'hyperfast',
      bloat: false,
      rawPower: true
    });
    
    return terminal.render(ctx.render());
  }
};
